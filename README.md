# Ygame

This is a text-based game where the objective is to avoid 'Y'.

Whether for Part 1 or Part 2, first build `Ysetting.cpp` into an application using `Ysetting.bat`, run it, and then click the 'Ygame' icon on your desktop.

## How to Play
 
You can control the game using the arrow keys or WASD. Avoid 'Y', earn points, and stay alive!
